Supported:

CRC-32 [ poly=0x04c11db7 init=0xffffffff xorout=0xffffffff ]
CRC-32/JAMCRC [ poly=0x04c11db7 init=0xffffffff xorout=0x00000000 ]

Not supported:

CRC-32/AUTOSAR
CRC-32/BZIP2
CRC-32C
CRC-32D
CRC-32/MPEG-2
CRC-32/POSIX
CRC-32Q
